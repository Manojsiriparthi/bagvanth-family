# Bagvanth Family Connect

A family connection platform to keep the Bagvanth family members connected and engaged.

## Features

- **Family Code Connection**: Connect with family members using unique family codes
- **Real-time Status**: See which family members are online
- **Family Tree Visualization**: View family relationships
- **Event Management**: Organize and track family events
- **Secure Connections**: Encrypted family communications

## Getting Started

1. Open `index.html` in your web browser
2. Enter your family code (e.g., `BAG001`)
3. Click "Connect" to join the family network
4. View connected family members and their status
